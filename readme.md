# TO DO APPLICATION

This application is used to manage your daily task with good user interface

_Technology used_

- python
- Fast API
- Prisma ORM
- Jinja 2

## Server start

```bash
pip install -r requirements.txt
prisma db push
uvicorn main:app
```
